Global Entrepreneurial Opportunity Perception Analysis

 Project Overview
This project analyzes global trends in perceived entrepreneurial opportunities using data from the Global Entrepreneurship Monitor (GEM) Adult Population Survey. The goal is to explore how opportunity perception has changed over time and identify countries with the largest shifts after 2020.


 Dataset
- Source: Global Entrepreneurship Monitor (GEM)
- Observations: 14,569
- Countries: 116
- Time period: 2001â€“2024
- Indicator analyzed: *Perceived entrepreneurial opportunities*

 Data Cleaning
- Removed rows with missing values in the observation column
- Selected relevant columns:
  - Country
  - Year
  - Indicator
  - Observed value

 Key Findings
- The global average perceived opportunity increased from **40.9% before 2020** to **54.5% after 2020**
- This represents a **33% relative increase**
- The top 5 countries with the largest post-2020 increases are:
  1. Thailand
  2. Indonesia
  3. Puerto Rico
  4. India
  5. Poland
- The largest observed increase was **41.23 percentage points**


Visualization
A bar chart was created to highlight the top 5 countries with the greatest increase in perceived entrepreneurial opportunities.
